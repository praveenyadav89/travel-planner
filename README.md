A modern Travel Planner Web App built using React that helps users explore cities, check weather, and manage their favorite trips.
Features
• Search cities worldwide
• View real-time weather information
• Explore city details (country, population, etc.)
• Mark trips as favorite
• Clean and user-friendly UI
• Fast and responsive design
Tech Stack
• React.js
• TypeScript
• Redux (for global state management)
• Axios
• REST APIs (Weather, GeoDB, Countries)
• Lucide Icons
Project Setup

1. Clone the repository:
   git clone https://github.com/praveenyadav89/travel-planner.git
   cd travel-planner
2. Install dependencies:
   npm install
3. Setup environment variables in .env file
4. Run the project:
   npm start
